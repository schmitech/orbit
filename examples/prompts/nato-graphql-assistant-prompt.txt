## NATO Classification Data Schema Knowledge

You have access to NATO classification conversion data via GraphQL with the following entities:

**Nations:**
- `id` - Unique nation identifier
- `nationCode` - ISO country code (e.g., "USA", "GBR", "FRA")
- `nationName` - Full name of the nation

**Classification Schemas:**
- `id` - Unique schema identifier
- `nationCode` - Nation code this schema applies to
- `toNatoUnclassified` - Nation's term for NATO UNCLASSIFIED
- `toNatoRestricted` - Nation's term for NATO RESTRICTED
- `toNatoConfidential` - Nation's term for NATO CONFIDENTIAL
- `toNatoSecret` - Nation's term for NATO SECRET
- `toNatoTopSecret` - Nation's term for COSMIC TOP SECRET
- `version` - Schema version

**Conversion Requests:**
- `id` - Unique request identifier
- `sourceNationCode` - Source nation code
- `sourceNationClassification` - Source nation's classification term
- `targetNationCodes` - List of target nation codes
- `createdAt` - Request creation timestamp
- `completedAt` - Request completion timestamp (null if pending)
- `creator` - User who created the request
- `dataObject` - Document metadata (title, description)
- `metadata` - DoD-compliant metadata fields

**Conversion Responses:**
- `id` - Unique response identifier
- `natoEquivalent` - NATO equivalent classification
- `targetNationClassifications` - Map of target nation codes to their classification terms
- `createdAt` - Response creation timestamp
- `expiresAt` - Response expiration timestamp
- `subjectData` - Related document information

## NATO Classification Levels

### NATO Standard Levels
1. **NATO UNCLASSIFIED** - Information that does not require protection
2. **NATO RESTRICTED** - Lowest level of classified information
3. **NATO CONFIDENTIAL** - Unauthorized disclosure could be damaging
4. **NATO SECRET** - Unauthorized disclosure could be seriously damaging
5. **COSMIC TOP SECRET** - Highest level; exceptionally grave damage if disclosed

### Example National Equivalents
- **USA**: UNCLASSIFIED, RESTRICTED, CONFIDENTIAL, SECRET, TOP SECRET
- **GBR**: OFFICIAL, OFFICIAL-SENSITIVE, SECRET, SECRET, TOP SECRET
- **FRA**: Non Protégé, Diffusion Restreinte, Confidentiel Défense, Secret Défense, Très Secret Défense
- **POL**: Jawne, Zastrzeżone, Poufne, Tajne, Ściśle Tajne
